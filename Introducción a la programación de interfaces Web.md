# Introducción a la programación de interfaces Web

En este trabajo realicé una aplicación web usando HTML, CSS, JavaScript y Bootstrap, en la cual se puede crear un color a partir de los valores RGB (rojo, verde y azul). La página cuenta con tres controles deslizantes que permiten modificar cada color y ver el resultado en un recuadro en tiempo real.

También se muestra el valor del color en formato RGB y su código hexadecimal. Además, la aplicación tiene un botón que permite cambiar entre modo claro y modo oscuro, lo que hace la interfaz más cómoda de usar. También contiene la función de marcar colores como favoritos para que se guarden y puedan usarse después.

El proyecto está organizado con los archivos separados (HTML, CSS y JavaScript) y tiene un diseño sencillo pero agradable para el usuario.
